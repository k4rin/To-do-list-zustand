# 📝 Todo List com Zustand

Uma aplicação simples de **lista de tarefas (Todo List)** desenvolvida com **React** e **Zustand** para gerenciamento de estado global.  
O objetivo do projeto é demonstrar uma forma **leve e simples de gerenciar estados** sem a complexidade de bibliotecas maiores.

---

## 🚀 Tecnologias Utilizadas

- ⚛️ React
- 🐻 Zustand
- 🎨 CSS
- ⚡ Vite

---

## 📦 Instalação

Clone o repositório:

```bash
git clone https://github.com/seu-usuario/nome-do-repositorio.git
```
Entre na pasta do projeto:

```bash
cd nome-do-repositorio
```
Instale as dependências:

```bash
npm install
```
Execute o projeto:

```bash
npm run dev
```
